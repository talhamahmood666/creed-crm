Verify Setup (OpenRouter)

The Verify button asks a free AI model to web-search each business and report whether it has its own website, confirm the phone, and find the owner name. Paste your OpenRouter key once — it is saved only in THIS browser, never inside the file.

Use a model that reliably supports tool calling, or web search won't run. Known-good free options: deepseek/deepseek-chat-v3-0324:free, meta-llama/llama-4-maverick:free, qwen/qwen3-235b-a22b:free. Avoid the random openrouter/free router — it sometimes lands on weak models that fake searches.
⚠ Anyone using this browser can read this key from storage. Set a spending cap on the key in your OpenRouter dashboard. Rotate it if it ever leaks. Always confirm the AI's answer against the "Maps" link — free models can be wrong or unsure.
